# EGX Light Trusted Analyzer

نسخة عملية خفيفة ودقيقة أكثر لأنها لا تستخدم أسعار وهمية.

## الفكرة
- الموقع لا يعرض بيانات عشوائية.
- التحليل يبدأ فقط بعد رفع CSV للأسعار.
- كل إشارة لها سبب وثقة وجودة بيانات.
- لو البيانات ناقصة، الثقة تنخفض أو تظهر "بيانات غير كافية".
